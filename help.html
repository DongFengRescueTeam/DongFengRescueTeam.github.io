# 东风救援平台使用手册

# 1. 家属端

# 1.1预填老人信息

进入首页→点击右上角我的图标，进入“我的”界面→点击中间的预填老人信息按钮，进入老人信息列表界面→上方添加老人信息按钮，进入老人详细信息界面→填写/修改老人的详细信息和上传老人的近期照片→点击下方的保存信息按钮进行老人信息的保存

# 1.2快速报案

进入首页→点击下方的我要报案按钮，进入快速报案界面→选择时间和地点，可选择已填写的老人或选择重新填写新信息→点击确认报案按钮，进入案件详细信息界面→立案成功，补全老人详细信息，点击下方保存按钮→信息保存成功

# 1.3案件管理

案件管理分为当前案件管理和历史案件管理。

# 当前案件管理：

进入首页→点击任意当前案件，进入案件详细信息界面→修改老人信息，查看案件进度（参与寻找的志愿者数量和线索信息），或点击下方的保存信息按钮/结案按钮→保存/结案完成

# 历史案件管理：

进入首页→点击右上角我的图标，进入“我的”界面→点击上方的我的历史案件按钮，进入历史案件列表界面→点击任意历史案件，进入进入历史案件详细信息界面→查看历史案件详细信息，点击下方立即报警按钮→跳转到拨号界面

# 1.4新线索提醒

进入首页→点击右上角我的图标，进入“我的”界面→点击下方的订阅我们的消息按钮，进入授权界面→同意授权→有新线索通知→查看新线索通知，点击下方进入小程序查看链接→进入对应的案件界面，查看详细线索通知

# 2. 志愿者端

# 2.1登陆

登陆界面→输入手机号码和密码，点击下方登陆按钮→登陆成功，进入案件列表界面

# 2.2查看案件列表

案件列表界面→查看具体的案件简要信息

# 2.3案件详细信息查看

案件列表界面→点击任意案件，进入案件详细信息界面→查看案件详细信息

# 2.4救援线索提供

案件列表界面→点击任意案件，进入案件详细信息界面→点击右下角相机图标，进入线索提交界面→选择时间和照片，填写线索内容，点击下方分享线索按钮

# 2.5对讲机功能

案件列表界面→点击任意案件，进入案件详细信息界面→点击右上角语音图标，进入对讲机界面→点击下方讲话按钮并讲话

# 2.6地图线索

案件列表界面→点击底部标签导航中间的地图线索图标→查看当前天气情况，自己和他人过去两小时的行走路线，参与案件的志愿者信息

# 3.Web管理端

# 3.1登陆

WEB端登陆界面→输入账号密码，点击下方登陆按钮→登陆成功，进入案件主页

# 3.2案件管理

案件主页→在左侧案件列表中选择任意案件的红色地图按钮→在右侧显示出该案件地图，并点击案件中心→显示出案件简要信息，点击左侧案件列表中案件的白色线索按钮→显示出该案件的线索详细信息

# 3.2地图查看

案件主页→在右侧地图上显示了所有的案件中心和当前在线的志愿者移动路线，鼠标滑动放大，鼠标移动到案件中心/志愿者图标→显示出案件简要信息/志愿者简要信息

# 3.3信息管理

信息管理分为志愿者信息管理，家属信息管理和老人信息管理三个部分。

# 志愿者信息管理：

案件主页→鼠标移动到左边的导航栏中间的眼睛图标→显示出志愿者管理，家属管理和老人管理三个部分，点击志愿者管理→进入志愿者信息管理界面，可新增，删除，查询，修改志愿者信息

# 家属信息管理：

案件主页→鼠标移动到左边的导航栏中间的眼睛图标→显示出志愿者管理，家属管理和老人管理三个部分，点击家属管理→进入家属信息管理界面，可新增，删除，查询，修改家属信息

# 老人信息管理：

案件主页→鼠标移动到左边的导航栏中间的眼睛图标→显示出志愿者管理，家属管理和老人管理三个部分，点击老人管理→进入老人信息管理界面，可新增，删除，查询，修改老人信息

# 3.4周边信息获取

周边信息获取分为医院信息获取和派出所信息获取

# 医院信息获取：

案件主页→点击左上方附近医院按钮→在最右侧显示出医院列表，上面有医院的联系方式和地理位置，点击任意医院→在右侧地图中显示出医院的位置

# 派出所信息获取：

案件主页→点击左上方附近派出所按钮→在最右侧显示出派出所列表，上面有派出所的联系方式和地理位置，点击任意派出所→在右侧地图中显示出派出所的位置

# 3.5大数据展示

案件主页→点击左边的导航栏下方的表格图标→点击做上方日期选择按钮，选择日期→可查看当前时间范围内的案件信息（案件总数、已找到、未找到、寻找中），丢失老人年龄分布统计，案件地区数量统计（地区案件地图，可选择对应地区查看），志愿者数量统计（志愿者数量的每日变化情况），丢失老人轮播图（播放丢失老人信息），案件类型统计（已找到、未找到、寻找中）
