# 使用帮助

# Android

激活

  1. 下载喵滴客户端(Android：2020.5版本以上)
  2. 登录或注册
  3. 用户信息->激活码激活->输入激活码
  4. 显示activated success即为激活成功
  5. 重启客户端刷新会员状态

# iOS

激活

  1. 下载喵滴客户端(iOS：2020.1.53版本以上)
  2. 登录或注册
  3. 用户信息->激活码激活->输入激活码
  4. 显示activated success即为激活成功
  5. 重启客户端刷新会员状态
