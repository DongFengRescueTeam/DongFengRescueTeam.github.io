# 2. 提供第三方接口

我们对外提供的第三方接口暂时有两大功能：

# 1. 获取没有完结的案件的详细信息.

# 2. 上传疑似照片帮助我们寻找走丢老人.

我们希望通过第三方接口来让社会各界机构以及其他app嵌入。从而为我们的公益事业添砖加瓦,提升社会影响。

接下来两个部分我们来介绍如何快速使用我们的第三方接口。

# 二. 第三方接口快速入门

# 2.1 注册账号

在注册网站上注册一个账号.按照页面提示完成必要信息填写。

# 2.2 申请密匙

在密匙网站看到申请密匙按钮。点击即可得到唯一密匙.

请注意:我们每一个账号都只允许申请一个唯一的密匙。在之后调用我们的接口时,需要将其作为必填参数来作为调用检测。所以希望用户们能够保管好该密匙.

# 2.3 使用接口

调用方式：通过http请求进行。

请注意:我们假设使用者对HTTP请求和API调用有一定的了解。如果您尚未了解这些前置知识，可以自行上网自学本内容。这里不展开论述

# 三. API文档

# 3.1 获取一个当前未结案的案件的详细信息接口

# 3.1.1 接口描述:

接口请求域名: 方法:get

"api.runningstars.cn:8080/External/getUnsolvedCaseInfos"

# 3.1.2 输入参数:

参数名称 必选 类型 描述
secretKey 是 String 公共参数,填写您申请到的密匙

# 3.1.3 输出参数:

参数名称 类型 描述
Success boolean 查询是否成功
Message String 结果信息 取值”成功”/”失败”
Code Long 结果代码 20000代表成功/20001代表失败
name String 老人名字
sex Int 老人姓名.0为女 1 为男
Age Int 老人年龄
dressHabit String 穿着习惯
Position String 丢失地点
Time String 丢失时间
familyTel Long 家属电话号码
familyName String 家属姓名
Urls List<String> 近期照片若干张
VolunteerNum Int 当前参与该案件的救援队员个数

# 3.1.4 示例:

输入示例:

  http://api.runningstars.cn:8080/External/getUnsolvedCaseInfos?secretKey=342251fb-6e44-4967-8f11-c3d993fd27c2

# 3.3 上传疑似图片相关接口

# 3.1.1 接口描述:

接口请求域名: 方法:get

”api.runningstars.cn:8080/External/upload”

# 3.1.2 输入参数:

参数名称 必选 类型 描述
secretKey 是 String 公共参数,填写您申请到的密匙
File 是 MultipartFile[] 图片数组，前提:form表单的enctype=mutipart/form-data

# 3.1.3 输出参数:

参数名称 类型 描述
Success boolean 查询是否成功
Message String 结果信息 取值”成功”/”失败”
Code Long 结果代码 20000代表成功/20001代表失败

# 3.1.4 示例:

输入示例:

  http://api.runningstars.cn:8080/External/Upload

输出示例:

  {    "success": true,    "code": 20000,    "message": "成功"  }
