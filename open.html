# 开放平台

实验功能，请勿过度依赖

喵滴的高级用户可以自己动手搭建功能，自行调用喵滴的开放接口 目前已经实现的开放能力有：添加文章

# 接入文档

  1. 向目标地址：https://sunshinesudio.club:8012/user/open/key 发送post请求，其中字段un为用户名，字段pwd为密码
curl --location --request POST 'https://sunshinesudio.club:8012/user/open/key' \
--form 'un="用户名"' \
--form 'pwd="密码"'
  1. 保存好喵滴返回给您的开放平台密码key,每次使用第一个步骤，key都会更新
//第一步返回的正确结果
{
    "info": "key:85H7f36H2Rv2u3fYk9vLiQz",
    "code": 200
}

其中85H7f36H2Rv2u3fYk9vLiQz为key。

  1. 添加文章
curl --location --request POST 'https://sunshinesudio.club:8012/page/open/add' \
--form 'key="开放码"' \
--form 'Title="标题"' \
--form 'Content="内容"' \
--form 'Book="书本名"' \
--form 'Chara="目录名"'

其中所有字段都必须填写,同一个书本以及目录下面，不能出现相同的文章标题。

返回添加成功结果

{
    "info": "xxxx",//文章id
    "code": 200
}
